Documentação do Projeto: Contagem de Animais em Vídeo Usando Python e YOLO
==========================================================================

Este projeto consiste em desenvolver uma ferramenta simples para a detecção e contagem de animais em vídeos usando python. Tal abordagem é baseada na biblioteca YOLO (You Only Look Once) e no framework Ultralytics. 

.. note::

   Este projeto é apenas um experimento e não será usado em produção.
